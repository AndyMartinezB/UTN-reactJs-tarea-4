# Tarea Nº4: Mi tarjeta de presentación en React

## Descripción del Proyecto

Este proyecto corresponde a la cuarta entrega del Curso de Desarrollo en React JS de
la UTN.BA. La tarea consiste en la creación de componentes funcionales, el uso de JSX y en particular el uso de props para personalizar contenido.

## Instrucciones

Para clonar el repositorio en tu terminal:

```bash
git clone https://github.com/AndyMartinezB/UTN-reactJs-tarea-4.git
```

Para ejecutar el proyecto (React + Vite):

```bash
npm install
npm run dev
```

Luego abrir el navegador en la URL indicada por Vite (por defecto `http://localhost:5173`).
